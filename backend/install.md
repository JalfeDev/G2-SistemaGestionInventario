# Guía de Instalación — Java + Spring Boot

## Requisitos previos
Antes de ejecutar el proyecto, asegúrate de tener instalado:
- Java JDK 25
- Maven 3.9+
- Git
- IDE recomendado:
  - IntelliJ IDEA
  - VS Code con Extension Pack for Java

---

# Instalación de Java

## Windows

1. Descargar JDK:
  https://www.oracle.com/java/technologies/downloads/#jdk25-windows

2. Instalar Java 25.

3. Verificar instalación:

```bash
java -version
